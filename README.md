# 🚀 FastX CLI


## ✨ 特性

- 🎯 开箱即用的开发体验
- 🔥 快速搭建项目框架
- 📦 优化的构建配置
- 🎨 现代化的开发工具链
- 🛡 TypeScript 支持
- 📱 响应式设计支持

## 🔧 安装

```bash
npm i lyb-fastx -g
```

## 🚀 快速开始

### 创建项目

```bash
# 创建新项目
fastx create my-app

# 进入项目目录
cd my-app

# 安装依赖
pnpm i

# 启动开发服务器
pnpm dev
```

## 🛠️ 可用命令

```bash
# 开发环境
pnpm dev

# 生产构建
pnpm build


## 🔍 环境要求

- Node.js >= 16.0.0
- pnpm >= 7.0.0
